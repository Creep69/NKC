Xtree für Jados
(C) Andreas Voggeneder

XT ist ein Dateimanager mit semigraphischer Oberfläche und ist dem Xtree-Pro aus DOS Zeiten nachempfunden.
Voraussetzung ist eine GDP64HS oder GDP64-FPGA!

Kurzanleitung:
Nach dem Start werden alle Dateien im aktuellen Laufwerk dargestellt.
Mit den Cursor-Tasten (CTRL-S/D/E/X) kann die Auswahl (weißes Rechteck) verschoben werden.
Befehle:
- t... Tag (CTRL-T): Markieren der aktuellen bzw. aller sichtbaren files
- u... Untag (CTRL-U): Aufheben der Markierung der aktuellen bzw. aller sichtbaren files
- c... Copy (CTRL-C): Kopieren der der aktuellen bzw. aller markierten files
- d... Delete (CTRL-K): Löschen der aktuellen bzw. aller markierten files
       Hier wird gefragt ob bei jeden file einzeln vor dem löschen rückgefragt werden soll oder nicht
- r... Rename (CTRL-R): Rename der aktuellen bzw. aller markierten files mittels 'Wildcards'
       Hierbei sind ? (ein zeichen) und * (alle nachfolgenden zeichen) möglich
       z.b. ??xy*.*
-l...  Log drive. Aktuelles laufwerk wechseln.
       z.B.: 0,1,A,B,...
-e     Edit file. Editor mit aktuellem file starten
-f     Filespecification z.b. "*.asm" oder "*.68k" oder a*.*
