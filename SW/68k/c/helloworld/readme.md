Minimale "Hello World" Routinen, die zeigen, daß prinzipiell mit C erzeugter Code auf dem NKC läuft.

Compilieren unter Windows/DOS mit

build HELLO.C

bzw.

build HELLO2.C

Das erzeugte *.68K File kann auf den NKC übertragen und unter Jados gestartet werden.

Jegliche Verbesserungen und Ergänzungen sind willkommen.

Es wäre toll, wenn hier eine Art kleines Tutorial zum C-Einstieg auf dem NKC entstehen könnte.
