NKC BUS3 v1.0 Neuauflage 27.4.2017

Layout neu erfaßt mit Sprint Layout 6 von Rene Vetter. Erweitert um einen Stromanschluß im 5mm Raster und ein Lochraster für kleinere Erweiterungen, z.B. Spannungswandler. Zum Betrieb mit 68000 16-Bit CPU müssen bei einem der beiden doppelreihigen Steckplätze die auf der Rückseite markierten Leiterzüge aufgetrennt werden.
